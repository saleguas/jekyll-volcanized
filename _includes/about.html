About this page

This page is left blank so you can fill the information. It has many features:

  • Particles.js implementation
  • Colored elements such as <\h1> and <\li> tags
  • Scrollspy with smooth scroll, as well as disabled navigation bars for that smooth look.
  • Card portfolio that can be customized and added to very easily
  • Very organized jekyll layout with easy swapping of elements.
  • AOS animations on various elements.
  • The classic solarized color pallete!
  • Fully responsive and mobile first!
  • And many other features with the _data directory that you can find on the github page!
